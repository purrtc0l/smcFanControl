smcFanControl Community Edition

Fan control for Intel Macs. Lightweight, modern, open source.

Monitor and control the fans in your Intel Mac. Set minimum fan speeds per application or create custom presets for different workloads. A community-maintained fork of the classic macOS utility, rebuilt with a modern interface and dramatically reduced footprint.
